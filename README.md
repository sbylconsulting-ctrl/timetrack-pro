# Cronómetro Pro

Aplicación web de fichaje y control horario multiempresa.

## Descripción

Cronómetro Pro es una app web hecha en HTML + React para gestionar:

- Inicio de sesión por empresa
- Roles de administrador y empleado
- Fichaje de entrada y salida
- Historial de registros
- Gestión de usuarios
- Exportación de datos en CSV

## Tecnologías usadas

- HTML
- CSS
- JavaScript
- React 18
- ReactDOM 18
- Babel Standalone
- LocalStorage

## Cómo usarla

1. Abre la aplicación en el navegador
2. Introduce el código de empresa
3. Inicia sesión con un usuario de prueba
4. Usa la app según tu rol:
   - **Empleado**: fichar entrada y salida, ver historial
   - **Administrador**: gestionar usuarios, revisar fichajes y exportar CSV
